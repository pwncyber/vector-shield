How to Use Vector Shield

System Requirements for Vector Shield

-Windows 7 Operating System or Higher
-4 Megabytes of Storage
-Latest Version of Java

Download Java

What To Do After Downloading

Once downloaded, you will have access to Vector Shield. You will be able to choose how secure you want your system to be and which settings you want to change. There are also advanced settings that you can see by clicking View in the top left corner of the application.

Three Levels of Security

There are three levels of security that Vector Shield offers. First, there is a low-level security option. This option is recommended for everyday users, and provides decent security with minimal impact on your computer's ease of use. Second, there is a mid-level security option. This option is recommended for enterprise buisiness environments, providing more security while moderately restricting usability. Third, there is a high-level security option. This option provides a extensive amount of security but a large impact on usability, as it disables services such as printing. These levels are indicated by buttons, which you can press in order to choose which one you want.

How to Implement Changes to Settings

In order to implement changes to advanced settings, you need to first choose which settings you want to change. Check the boxes that correspond with the actions you wish Vector Shield to take, then click the 'Secure my System' button. Using advanced settings is for users that wish to fully control the actions that Vector Shield takes, and is not recommended if you dont understand the choices. Choosing a security option mentioned previously will set advanced settings for you, and no furthur action other than clicking 'Secure my System' is needed.

Impact of Vector Shield

Vector Shield changing your computer's settings will impact services and applications that rely on any disabled features. Dont worry though, as long as you choose the right security option for you, the effects are minimal.

For the low option, possible effects includes:
Insecure connections being blocked due to stricter encryptions, services relying on insecure legacy protocols not working, being prompted by basic windows security responding to possible threats, defualt guest & admin accounts being restricted, inability to access and control your computer remotely.

For the medium option, possible effects include low option effects, as well as:
Certain administrative actions being blocked due to restrictions on low level users, restrictions on removable media, services such as peer networking , Windows Maps, and Mobile Hotspot not working due to network and location sharing services being disabled, furthur restrictions on remote access.

For the high option, possible effects include medium option effects, as well as:
Inability to restore files due to heavy restrictions on low level users, basic windows services such as Microsoft store not working due to such services being disabled, inability to print as it is disabled as well as Bluetooth and Xbox services.
